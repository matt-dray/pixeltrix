A work-in-progress R package for interactive pixel art in the plot window.

1. Provide a matrix with 0s and 1s in it.
2. It will be plotted as an image.
3. Click squares to convert them from 'off' to 'on' and vice versa.
4. Press <kbd>Esc</kbd> when you're done.
5. Your modified matrix is returned.
