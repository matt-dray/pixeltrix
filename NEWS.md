# pixeltrix 0.0.0.9003

* Added argument `n_states` to `click_pixel()` that lets users choose the number of 'states' that a pixel can take (#4).
* Added argument `grid` to `click_pixel()` that lets users turn on or off a grid overlay (#5).
* Changed colours to greyscale.

* Added GitHub Actions for R-CMD check, tests, {pkgdown} webbite.
